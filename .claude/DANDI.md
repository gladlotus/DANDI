### 3. 가격 산정 공식 (Pricing Logic)

[총 견적] = 기본 작업료 + 시간 공임비 + 재료비 (이후 100원 미만 무조건 내림)

1) 기본 작업료: 건당 고정 2,000원
2) 시간 공임비 (총 출력 시간 기준):
   - 첫 1시간: 무료 (0원)
   - 초과 시간: 초과 1시간당 1,000원 (올림 계산)
   - 예: 50분 -> 0원 / 1시간 24분 -> 1시간 초과로 +1,000원 / 2시간 08분 -> 2시간 초과로 +2,000원
3) 재료비 (필라멘트 사용량 기준):
   - 기본 라인업 (PLA Basic/Matte White): g당 50원
   - 지정 색상 옵션 (Black, Gray 등 요청 시): 기본 견적에 +2,000원 고정 세팅비 추가
4) 십 원 단위 내림 (절사) 규칙: Math.floor(금액 / 100) * 100 (예: 9,275원 -> 9,200원)

### 5. 웹 페이지 UI/UX 및 스타일링 지침 (★ Glassmorphism 반투명 스타일 ★)

1) 배경 및 레이아웃 구조
   - 전체 배경: 깊고 우아한 남색 (Dark Navy / Deep Slate Navy)
   - 배경 패턴: 건축 도면 느낌의 은은한 격자 무늬 청사진 그리드 (Blueprint Grid Pattern)
   - 전체 컨테이너: 화면 중앙(max-w-3xl, mx-auto) 수직 배치

2) 카드 디자인
   - 카드 배경색: **은은한 투명도가 있는 반투명 유리 카드 스타일 (예: rgba(255, 255, 255, 0.08~0.12) / backdrop-filter: blur(12px))**
   - 카드 테두리: 은은한 하얀 빛의 미세 테두리 (`border: 1px solid rgba(255, 255, 255, 0.18)`)
   - 카드 텍스트: 남색 배경 위 투명 카드에서 명확히 읽히는 선명한 밝은 톤 (White / Soft Cyan / Pale Gray)
   - 카드 정렬: 카드 내부 요소들은 좌측 정렬(text-left)을 기본으로 정돈된 인상을 주도록 배치

3) 화면 구성 및 Step 카드 구조
   ■ 상단 헤더: 최상단 중앙 한글 "단디" 로고 + 메인 타이틀 "3D 프린터 예상 견적 계산기"
   ■ 버튼: ['단디' 알아보기] (클릭 시 브랜드 설명 및 가격 정책 펼쳐짐)

   ■ STEP 1: 모델 업로드
     - STL 드래그앤드롭 업로드 영역 (업로드 시 크기/부피/면적 자동 감지)
     - 수동 입력 탭 (시간/g수 직접 입력 전환 기능)

   ■ STEP 2: 조건 설정 (간결한 버튼 형태)
     - 서포트 필요 여부 선택 (★ 핵심 오차 보정 옵션 ★):
       * [ 서포트 없음 (일반 매스/지형) ] -> 기본 가중치 (1.0x)
       * [ 서포트 필요 (공중 돌출/캔틸레버 구조) ] -> 선택 시 계산된 예상 시간 및 재료 무게에 각각 +15% 가중치 자동 보정 적용 (1.15x)
     - 재질/컬러 선택: [ Basic White (유광) ] / [ Matte White (무광) ] / [ 지정 색상 (+2,000원) ]
     - 출력 품질 선택: [ ⚡ 0.24mm Draft ] / [ ⚖️ 0.20mm Standard ] / [ ✨ 0.16mm High Quality ]

   ■ STEP 3: 예상 견적 확인
     - 256mm 초과 시: 크기 초과 경고 알림
     - 정상 범위 내:
       1) 핵심 정보 요약 위젯 (📦 예상 재료량: OOO g / ⏱️ 예상 출력시간: OO시간 OO분 / 💰 최종 예상비용: O,OOO 원 - 민트/시안 컬러 강조)
       2) 비용 세부 내역 (기본 작업료 2,000원 + 시간 공임비 + 재료비)
       3) "십 원 단위 내림 혜택 적용됨" 배지/태그
       4) [ 1:1 카카오톡 상담 / 주문하기 ] 액션 버튼
