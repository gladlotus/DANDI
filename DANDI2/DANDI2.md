### 2. 빌드 플레이트 크기 제한 & 초과 경고 로직

* 보유 장비: Bambu Lab A1 (0.4mm 노즐)
* 최대 출력 가능 크기: 256mm x 256mm x 256mm (X, Y, Z)

[크기 검증 처리 규칙]
- 업로드된 STL 파일의 Bounding Box(가로 X, 세로 Y, 높이 Z) 중 어느 한 축이라도 256mm를 초과하는 경우:
  1) 견적 산출을 중단함
  2) Step 3 카드 영역에 경고 메시지 표시:
     "⚠️ 모델의 크기(256mm 초과)가 3D 프린터 출력 범위를 초과했습니다. 모델을 분할(Split)하여 다시 업로드하시거나 1:1 채팅으로 문의해 주세요!"

### 3. 가격 산정 공식 (Pricing Logic)

[총 견적] = 기본 작업료 + 시간 공임비 + 재료비 (이후 100원 미만 무조건 내림)

1) 기본 작업료: 건당 고정 2,000원
2) 시간 공임비 (총 출력 시간 기준):
   - 첫 1시간: 무료 (0원)
   - 초과 시간: 초과 1시간당 1,000원 (올림 계산)
   - 예: 50분 -> 0원 / 1시간 24분 -> 1시간 초과로 +1,000원 / 2시간 08분 -> 2시간 초과로 +2,000원
3) 재료비 (필라멘트 사용량 기준):
   - 기본 라인업 (PLA Basic/Matte White): g당 50원
   - 지정 색상 옵션 (Black, Gray 등 요청 시): 기본 견적에 +2,000원 고정 세팅비 추가
4) 십 원 단위 내림 (절사) 규칙: Math.floor(금액 / 100) * 100 (예: 9,275원 -> 9,200원)

### 5. 화면 구성 및 Step 카드 구조 (UI/UX 지침)

■ STEP 1: 모델 업로드
- STEP 1 제목 우측 끝에 [안내 보기] 버튼 배치 (화살표 회전 아이콘 포함 아코디언 토글 방식)
- [안내 보기] 클릭 시 카드 내부 제목 아래로 펼쳐지는 안내 내용:
  · STL 파일(바이너리/ASCII 모두 지원)을 업로드하면 크기·부피·표면적을 자동 분석합니다.
  · 최대 출력 가능 크기는 256 × 256 × 256mm입니다. 이를 초과하면 견적이 산출되지 않습니다.
  · 시간/무게를 이미 알고 있다면 '수동 입력' 탭에서 직접 값을 넣어도 됩니다.
- STL 드래그앤드롭 업로드 영역 및 수동 입력 탭 제공

■ STEP 2: 조건 설정
- STEP 2 제목 우측 끝에 [스펙 보기] 버튼 배치 (화살표 회전 아이콘 포함 아코디언 토글 방식)
- [스펙 보기] 클릭 시 카드 내부 제목 아래로 펼쳐지는 안내 내용:
  · Basic/Matte White: 기본 라인업 PLA, 추가 비용 없음 (g당 50원)
  · 지정 색상: Black, Gray 등 별도 색상 요청 시 세팅비 +2,000원 고정 추가
  · ⚡ Draft(0.24mm): 가장 빠른 출력, 표면이 다소 거칠 수 있음
  · ⚖️ Standard(0.20mm): 속도와 품질의 균형, 기본 권장 옵션
  · ✨ High Quality(0.16mm): 가장 정교한 표면, 출력 시간 증가
- 조건 선택 버튼 그룹:
  1) 서포트 필요 여부: [ 서포트 없음 (기본) ] / [ 서포트 필요 (시간/무게 보정 적용) ]
  2) 재질/컬러 선택: [ Basic White ] / [ Matte White ] / [ 지정 색상 (+2,000원) ]
  3) 출력 품질 선택: [ ⚡ 0.24mm Draft ] / [ ⚖️ 0.20mm Standard ] / [ ✨ 0.16mm High Quality ]

■ STEP 3: 예상 견적 확인
- 256mm 초과 시 크기 초과 경고 알림 모달/카드 표시
- 정상 범위 내:
  1) 핵심 정보 요약 위젯 (📦 예상 재료량: OOO g / ⏱️ 예상 출력시간: OO시간 OO분 / 💰 최종 예상비용: O,OOO 원 - 가장 크고 명확하게 강조)
  2) 비용 세부 내역 (기본 작업료 2,000원 + 시간 공임비 + 재료비)
  3) "십 원 단위 내림 혜택 적용됨" 배지/태그
  4) [ 1:1 카카오톡 상담 / 주문하기 ] 액션 버튼
