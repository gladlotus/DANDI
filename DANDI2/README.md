# 단디(DANDI) · 3D 프린터 예상 견적 계산기

단국대학교 교내 3D 프린팅 출력 대행 서비스 '단디'의 웹 견적 계산기입니다.

STL 파일을 업로드하면 크기·부피·표면적을 분석해 예상 재료량, 출력 시간, 최종 견적을 자동으로 계산합니다.

## 특징

- **외부 의존성 없음** — HTML 파일 하나로 동작합니다. 빌드 도구·서버·라이브러리가 필요 없습니다.
- **STL 파서 자체 구현** — 바이너리/ASCII 형식 모두 지원합니다.
- **실측 기반 추정** — Bambu Lab A1의 실제 슬라이싱 데이터 5종 × 3품질(15개 지점)에 맞춰 계수를 보정했습니다.

## 추정 정확도

실측 대비 평균 오차:

| 항목 | 평균 | 최대 |
|---|---|---|
| 모델 무게 | 0.8% | 2.1% |
| 서포트 무게 | 2.1% | 4.6% |
| 출력 시간 | 0.7% | 3.4% |

> 인필 0%로 슬라이싱한 모델은 내부 채움을 건너뛰어 g당 시간이 유독 빠릅니다. 일반적인 인필 조건과 특성이 달라 시간 보정에서 제외했으며, 이 경우 실제보다 길게 추정됩니다.

## 가격 정책

- 기본 작업료: 건당 2,000원
- 시간 공임비: 시간당 1,000원 (첫 1시간 무료)
- 재료비: g당 50원
- 지정 색상: 세팅비 +2,000원
- 100원 미만 내림

## 장비

Bambu Lab A1 · 0.4mm 노즐 · 최대 출력 크기 256 × 256 × 256mm

## 파일 구성

| 파일 | 설명 |
|---|---|
| `index.html` | 배포용 (GitHub Pages 등이 자동으로 찾는 진입점) |
| `DANDI2.html` | 작업용 사본 (내용은 `index.html`과 동일) |
| `dandi-display.woff2` | 로고 전용 서체 서브셋 (5KB) — **배포 시 반드시 함께 올려야 합니다** |
| `og-image.png` | 카카오톡·SNS 공유 시 표시되는 미리보기 이미지 |
| `DANDI2.md` | 기획 명세 및 실측 데이터 |

### 서체

로고 '단디'에는 **RixInooAriDuri**를 사용합니다. 원본(929KB)에서 실제 쓰는 글자만 추출해 **5KB(99.4% 감소)** 로 서브셋했으며, 저장소에는 서브셋만 포함되어 있습니다.

로고 문구를 바꾸는 등 서브셋을 다시 만들어야 하면 원본 TTF를 구해 아래 명령을 실행하세요:

```bash
pip install fonttools brotli
pyftsubset "RixInooAriDuri Regular.ttf" --text="단디" \
  --output-file=dandi-display.woff2 --flavor=woff2
```

## 로컬에서 실행

```bash
python -m http.server 8000
```

브라우저에서 `http://localhost:8000` 으로 접속합니다.

## 배포

정적 호스팅이면 어디든 올릴 수 있습니다 (GitHub Pages, Netlify, Cloudflare Pages 등).

배포 후 `index.html`의 `og:image`를 **절대 주소**로 바꿔야 카카오톡 공유 미리보기가 정상 표시됩니다.

```html
<meta property="og:image" content="https://내주소/og-image.png">
```
